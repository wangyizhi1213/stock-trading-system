# stock-trading-system 模拟股票交易系统 （实现游客和用户的各种功能）
    该项目基于Vue2.X的框架搭建的，需要搭配模拟后端实现。
    前端npm run serve的时候保持模拟后端一直运行。
    记得修改vue.config.js文件，以正确链接后端监听窗口。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
    代码完成后终端运行以上命令，若成功生成localhost：8080之类的链接，点进链接就能看到网页。

### Compiles and minifies for production
```
npm run build
```
终端执行以上命令保证一些必要包的下载。
